# Manual de Supervivencia Gamer

## Descripción
Este proyecto es una página web desarrollada en HTML que funciona como una guía para mejorar en videojuego: League of Legends.

## Contenido
- Página principal:
    *En esta pagina se podran ver diferentes consejos para subir de elo o por lo menos no estar tan "stucked"
- Consejos para jugadores
    *En esta sección doy consejos, de los mas "faciles" o los mas comunes que se tienen que tener en cuenta a la hora de subir elo de la forma mas rapida y eficiente posible.
- Herramientas utiles:
    *En esta seccion es una tabla en la cual explico las mejores herramientas que se pueden utilizar como para jugar y obtener la mayor ventaja. Y  tmb recomiendo el utilizar youtube como para buscar guias tanto del juego, como del personaje en especifico que quieras jugar
- Imágenes:
    *En esta pagina, seleccione 2 imagenes, para usar de muestra, una es de un equipo competitivo que esta analizando que personaje seleccionar para jugar la partida, la otra imagen es del minimapa del juego, el cual es muy util para tomar decisiones de que hacer. Y el video es el mejor video que explica una de las mecanicas mas improtantes del juego.
- Formulario de contacto
    *La idea de esta seccion es que la persona que necesite mande un mensaje para encontrar la respuesta a la duda que tenga.
